MailChimp for GigPress

== Description ==
MailChimp for GigPress WordPress plugin is developed by [The Right Software](https://therightsw.com/) web development company. It requires that GigPress be installed already on the WordPress. Then you can select GigPress events into this plugin and select MailChimp template to send to users. The MailChimp for GigPress WordPress plugin selects upcoming events into an HTML and inserts into the MailChimp template body using mc:edit. The campaign will be sent immediately. You must own a valid MailChimp account to use this plugin.

Note: We are independent from GigPress and MailChimp. We did this for a client and then made a small present for all WP users.
